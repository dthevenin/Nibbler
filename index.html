Nibbler

Bender / VS Toolkit wrapper

Nibble Page

Nibbler is a wrapping of VSToolkit into Bender framework:
  • VSToolkit ia HTML5 framework, created to easily develop Hybrid Applications for mobile devices.
  • Bender is an experimental, declarative Web application.
